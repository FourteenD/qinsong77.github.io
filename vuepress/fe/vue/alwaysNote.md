## Usage

```javascript
module.exports = {
  plugins: ['@vuepress/active-header-links']
}
```

### Passing Options

```javascript
module.exports = {
  plugins: ['@vuepress/active-header-links', {
    sidebarLinkSelector: '.sidebar-link',
    headerAnchorSelector: '.header-anchor'
  }]
}
```
